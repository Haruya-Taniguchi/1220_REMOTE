1220_REMOTO
====
このコードは自分の学生寮の部屋の照明をスマートフォンで操作するためのソースコードです。ESP8266にブラウザからアクセスし、表示されたボタンを押して照明を操作します。
まだ完成はしていないので使用方法などの情報は今後追加していく予定です。
## Depandency
- 使用言語：
	- Arduino
	- HTML
	- CSS
	- JavaScript
- 実行環境
	- ESP8266搭載ボード
	- Google Chrome、Safari
- 開発環境：
	- Arduino IDE 1.8.5
	- Atom
